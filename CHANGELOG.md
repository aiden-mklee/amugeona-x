# Changelog

## v1.0.1 (2026-06-19)

### 개선
- **룰렛 카드 높이 안정화** — 스피닝 중 이름 영역을 고정 높이(64px) 컨테이너로 감싸 세로 가운데 정렬. 이름이 한 줄이든 두 줄이든 카드 크기가 일정하게 유지됨
- **스피닝 힌트** — 카드 하단에 "결정하는 중…" 텍스트를 깜빡임 애니메이션으로 표시
- **결정 카드 간격 조정** — 카테고리·링크 상하 여백 조정으로 스피닝 카드와 높이 근사 매칭
- **특수 카드 UI 정리** — 배달·편의점·구내식당 stamp 칩 제거, 이모지를 이름에 직접 포함
- **학식 카드** — stamp 칩 제거, 카테고리 텍스트("학식") 제거로 특수 카드와 동일한 구조

### 추가
- **구내식당 특수 뽑기** — 배달·편의점에 이어 🏢 구내식당 항목 추가 (뽑기 풀에만 포함, 목록 미노출)

## v1.0.0 (2026-06-19)

첫 번째 정식 버전.

### 핵심 기능
- **GPS 기반 주변 식당 검색** — 카카오맵 JS SDK (services 라이브러리, 지도 미사용)
- **거리 프리셋** — 도보 5분(350m) / 도보 10분(700m) / 차 5분(3km) / 차 10분(6km)
- **랜덤 45개 풀** — 3개 오프셋 중심점에서 병렬 검색 후 머지·셔플·거리순 정렬
- **오늘 점심 정해줘** — 풀에서 랜덤 1곳 결정, 카카오맵 링크 연결
- **지역 검색** — 수동으로 다른 지역 설정 가능 (후보 목록 드롭다운)
- **학식 표시** — 경복대 남양주캠퍼스 1.5km 이내 시 피오니·아질리아 상단 노출

### 야간 모드 (🌙)
- 세그먼트 컨트롤(☀️/🌙)로 점심·저녁 모드 전환
- 야간 테마 — 다크 네이비 배경, 옐로우(#ffd166) 포인트 컬러
- 술집·포차 키워드 병렬 검색으로 풀 확장
- 저녁 카테고리(술집·호프·포차·이자카야·바·펍·포장마차·맥주·요리주점·칵테일 등) 3배 가중치 샘플링
- 간식 카테고리 자동 제외
- 학식 숨김

### 디자인
- **폰트** — Pretendard (본문), Black Han Sans (히어로 타이틀)
- **결정 카드** — slam 애니메이션 (scale + translateY), 오렌지 그림자
- **CSS 변수** — `--accent`, `--app-bg`, `--card-bg`, `--text-*` 등으로 테마 전환 구조화
- 모바일 패딩 최적화 (좌우 20px)
