# Changelog

## v1.0.0 (2026-06-19)

첫 번째 정식 버전.

### 핵심 기능
- **GPS 기반 주변 식당 검색** — 카카오맵 JS SDK (services 라이브러리, 지도 미사용)
- **거리 프리셋** — 도보 5분(350m) / 도보 10분(700m) / 차 5분(3km) / 차 10분(6km)
- **랜덤 45개 풀** — 3개 오프셋 중심점에서 병렬 검색 후 머지·셔플·거리순 정렬
- **오늘 점심 정해줘** — 풀에서 랜덤 1곳 결정, 카카오맵 링크 연결
- **지역 검색** — 수동으로 다른 지역 설정 가능 (후보 목록 드롭다운)
- **학식 표시** — 경복대 남양주캠퍼스 1.5km 이내 시 피오니·아질리아 상단 노출

### 야간 모드 (🌙)
- 세그먼트 컨트롤(☀️/🌙)로 점심·저녁 모드 전환
- 야간 테마 — 다크 네이비 배경, 옐로우(#ffd166) 포인트 컬러
- 술집·포차 키워드 병렬 검색으로 풀 확장
- 저녁 카테고리(술집·호프·포차·이자카야·바·펍·포장마차·맥주·요리주점·칵테일 등) 3배 가중치 샘플링
- 간식 카테고리 자동 제외
- 학식 숨김

### 디자인
