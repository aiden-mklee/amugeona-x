# CLAUDE.md

Claude Code가 이 프로젝트를 이어서 작업할 때 참고하는 메모.

## 한 줄 요약
GPS로 주변 식당을 찾아 랜덤으로 한 곳을 "결정"해주는 점심 앱. 지도는 안 그리고
카카오맵 링크로 연결만 한다. React 18 + Vite, 외부 상태관리 라이브러리 없음.

## 작업 규칙
- **수정·삭제 전 반드시 확인.** 파일, 코드, 설정을 수정하거나 삭제하기 전에 항상 먼저 물어본다. 허락 없이 바로 변경하지 않는다.
- **모르는 것은 무조건 질문.** 의도가 불명확하거나 판단이 서지 않으면 짐작하지 말고 질문한다.
- **디자인 관련 작업은 Q&A 방식으로.** 디자인 개선·변경 사항은 먼저 방향을 제안하고 답변을 받은 뒤 진행한다. 임의로 스타일을 바꾸지 않는다.

## 핵심 규칙
- **지도 SDK(지도 그리기) 쓰지 말 것.** services 라이브러리만 동적 로드해서 검색만 한다 (`src/lib/kakao.js`). REST API 직접 호출은 CORS로 막히므로 금지.
- **GPS가 기본, 지역검색은 서브.** 진입 시 항상 GPS를 먼저 시도하고, 지역검색은 접힌 보조 입력으로만 노출한다. 이 우선순위를 깨지 말 것.
- 카카오 키는 `.env` 의 `VITE_KAKAO_JS_KEY` 로만 읽는다. 코드에 하드코딩 금지.
- 거리 표시는 직선거리 기반 추정이라 "약 N분"으로만 표현한다 (정확한 도보거리 아님).

## 자주 건드릴 곳
- 학식/캠퍼스: `src/config/places.js` (`GYEONGBOK_NYJ`, `CAFETERIAS`, `CAMPUS_RADIUS_M`)
- 거리 프리셋: 같은 파일 `RADIUS_PRESETS`
- 음식 카테고리: `src/lib/kakao.js` 의 `FD6`(음식점). 카페 추가하려면 `CE7` 병행 검색.
